Workshop 7

In the last workshop, we looked at inputs with text blocks and noticed that the code we used was encapsulated in a function that was called at the click of a button. A function in JavaScript is a reusable block of code designed to perform a specific task. Functions are fundamental in making your code modular, easier to read, and reusable.

Declaring a Function

In JavaScript, a function is defined using the function keyword, followed by the function name and a pair of parentheses (). Hereâ€™s a simple example:

Example:


function greet() {
    console.log("Hello, World!");
}

To call the function and run the code inside it:


greet(); // Outputs: "Hello, World!"

Function Parameters and Arguments

Functions can accept parameters, allowing you to pass in values for them to use. You can think of parameters as placeholders for the actual values that will be provided when the function is called.

Example:


function greetUser(name) {
    console.log("Hello, " + name);
}

greetUser("Dave");  // Outputs: Hello, Dave

<input type="text" id="nameParameter" placeholder="Enter your name">
<button onclick="greetUserParam()">Greet Me!</button>

<script>
function greetThisUser(name){
    document.getElementById("greetingParamText").innerText = "Hello, " + name + "!";
}

function greetUserParam() {
    greetThisUser(document.getElementById("nameParameter").value);
}
</script>

Return Values

A function can return a value using the return statement. The return value can be captured and used in other parts of your code.

Example:


function square(x) {
    return x * x;
}

let result = square(5);
console.log(result);  // Outputs: 25

Arrow Functions

Arrow functions provide a shorter syntax for writing functions. These are often used for shorter, inline functions.

Example:


const multiply = (a, b) => a * b;

console.log(multiply(2, 3));  // Outputs: 6

Function Expressions

Function expressions allow you to assign a function to a variable. This function can be anonymous, meaning it doesn't need a name.

Example:


const sayHi = function() {
    console.log("Hi!");
};

sayHi();  // Outputs: Hi!
